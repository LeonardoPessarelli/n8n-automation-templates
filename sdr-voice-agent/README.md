# Agente de Voz SDR

Agente de prospecção outbound via ligação de voz automatizada usando Vapi. Liga para leads, apresenta a solução e agenda reunião na agenda do responsável.

## Workflow

| Arquivo | Função |
|---|---|
| `Agente-de-Voz-SDR.json` | Recebe lead, aciona ligação via Vapi, registra resultado |

## Integrações

- Vapi (ligação de voz com IA)
- n8n (orquestração e registro de resultado)
